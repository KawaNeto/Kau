Lista -- Exercício 3

“Qual é a diferença entre linguagem de programação Server Side e Client Side? Cite exemplos de cada tipo”.
Resposta: É o processamento que acontece no servidor, antes que o conteúdo chegue ao navegador do usuário. O servidor recebe a requisição (por exemplo, quando você acessa um site), processa dados, acessa bancos de dados e gera o resultado (geralmente um HTML pronto) para enviar ao cliente.
